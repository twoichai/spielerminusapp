Einleitung

Willkommen zur Nutzer Dokumentation. Diese Seite wird Ihnen helfen, sich mit den Funktionen und der Bedienung unseres Produkts vertraut zu machen.

Installation

Um die Software zu installieren, folgen Sie diesen Schritten:

  1. Laden Sie das Installationspaket von unserem Github herunter.
  2. Führen Sie das Installationsprogramm aus und folgen Sie den Anweisungen in der ReadME
  3. Starten Sie die Anwendung nach der Installation mithilfe der ReadMe des Porjektes

Verwendung

Nachdem die Software installiert ist, können Sie sie wie folgt verwenden:

Grundlegende Bedienung

Starten Sie die Anwendung und melden Sie sich mit Ihrem Benutzernamen und Passwort an. Das Hauptmenü bietet Ihnen Zugriff auf alle wichtigen Funktionen.

Beim ersten anmelden als Admin ist die Änderung des Passwortes notwendig.

Erweiterte Funktionen

Für erweiterte Funktionen, verwenden Sie das Menü Einstellungen und passen Sie die Optionen nach Ihren Bedürfnissen an.

Support

Wenn Sie Hilfe benötigen, kontaktieren Sie unseren Support unter spaten@support.de oder besuchen Sie unsere Support-Seite.
